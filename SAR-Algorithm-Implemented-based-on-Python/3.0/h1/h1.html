Theoretical Background¶

1.FM:¶

s(t) = rect($\frac{t-t_0}{T}$)exp(j $\pi$ K $ (t-t_0)^2$ )

2.Matched Filtering:¶

h(t) = rect($\frac{t}{T}$)exp(-j $\pi$ K $t^2$ )

$s_{out}$(t) = conv(s(t),h(t)) $\approx$ $Tsinc(KT(t-t_0))$
